# 好哇！按鈕 🎉

一個可愛的互動式網頁應用程式，點擊就會播放「好哇！」的聲音。

## 功能
- 🔊 隨機播放 5 種不同的「好哇」音效
- ✨ 點擊時有粒子噴發特效 (emoji)
- 💬 浮動「好哇！」文字特效 (IG/TikTok 風格)
- 📊 計數器記錄點擊次數
- 📤 一鍵分享功能
- 📱 支援 PWA，可安裝到手機桌面
- 📴 離線也能使用

## 線上體驗
https://wowhugo.github.io/hoawa/

## 本地開發
```bash
cd app
npm install
npm run dev
```

## 技術棧
- React + Vite
- PWA (Service Worker)
- CSS Animations
